# ポインタ
ショートカット

# アドレス
変数につけられた番号。

# &
&は変数のアドレスを求める演算子。

# ポインタ型
指定アドレスに記憶された数値を取り出すため、アドレスを記憶する変数の型。
どんな型の変数のアドレスだったかわかる。

# ポインタ値
変数のアドレスの値。

# ポインタ変数
ポインタ型で宣言された実際の変数。
変数のアドレスを自由に代入できる。
アドレスのメモリを読んだり書き換えたりできる。

# 宣言
型名 *変数名;
int *p;

# アドレスを代入する
型名 *変数名;
ポインタ変数 = &変数
int *p
int i;
p = &i;
