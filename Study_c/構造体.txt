# 構造体
異なる型をまとめて扱う。

## 宣言
struct 構造体の型名 構造体変数;

## アクセス（使い方）
構造体変数名.要素名


# 構造体型
構造体タグを直接新しい型にする場合。

## 宣言
typedef 新しい型の形 新しい型名


# 構造体タグと構造体型を1度に宣言
typedef struct {
    データ
} 構造体の型名;

# 引数として渡す
void 関数名(構造体の型名 構造体変数)

全ての値がコピーされる。
受け取る側の関数で引数の中身を変更しても、元の構造体変数には影響しない。
